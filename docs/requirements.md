## Functional Requirements

Following are the functional requirements of the Inventory Management System:

### Wholesaler (Admin)

| ID | Requirement Description |
|----|--------------------------|
| FR1 | The wholesaler should be able to **register** and **log in** securely. |
| FR2 | The wholesaler should be able to **add new products** with details such as name and stock quantity. |
| FR3 | The wholesaler should be able to **edit or delete existing products**. |
| FR4 | The wholesaler should be able to **update product stock** and **record transactions** after a purchase or sales. |
| FR5 | The wholesaler should be able to **record transactions** for both **sales** and **purchases**. |
| FR6 | The wholesaler should be able to **view all transactions**, including those made by salespersons. |

### Salesperson

| ID | Requirement Description |
|----|--------------------------|
| FR10 | The salesperson should be able to **regiser** and **log in** securely. |
| FR11 | The salesperson should be able to **view all available products** and their stock quantities. |
| FR12 | The salesperson should be able to **record sales transactions only**. |
| FR13 | The salesperson should be able to **view only their own transaction history**. |
